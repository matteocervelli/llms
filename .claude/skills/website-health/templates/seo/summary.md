# SEO — L1 Summary

## Critical Checks (per page)

1. **Title tag** — 50-60 chars, unique, contains primary keyword
2. **Meta description** — 150-160 chars, unique, compelling
3. **H1 tag** — Exactly one per page
4. **Canonical** — Self-referencing `<link rel="canonical">`

## Site-Wide Checks

5. **robots.txt** — Not blocking important pages or sitemap
6. **sitemap.xml** — Valid XML, URLs return 200
7. **Hreflang** — Language cross-references (bilingual sites)
8. **Open Graph + Twitter Cards** — og:title/desc/image/url + twitter:card
9. **Structured data** — JSON-LD schema.org on key pages
10. **RSS/Atom feed** — Valid, recent entries
